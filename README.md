# Coca-Cola Landing Page
 
## :rocket: Tecnologias
Esse projeto foi desenvolvido com as seguintes tecnologias:
* HTML5
* CSS3
* JavaScript

## :computer: Projeto
Este projeto apresenta uma landing page com 4 sabores diferentes de Coca-Cola:
* Coca - Cola Original
* Coca - Cola Zero
* Coca - Cola Life
* Coca - Cola Cherry
